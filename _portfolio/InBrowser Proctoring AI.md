# InBrowser Proctoring AI

## Overview
A comprehensive AI-powered exam monitoring solution designed to ensure academic integrity during online examinations. The system uses detection techniques to monitor test-takers in real-time.

## Key Features
- Facial recognition and verification using DeepFace
- Gaze tracking to monitor attention and focus
- Object detection for prohibited items (books, mobile devices)
- Real-time monitoring and alert system
- Browser-based implementation for easy access

## Technologies Used
- DeepFace (Facial Recognition)
- YOLOv8 (Fine-tuned for object detection)
- Gaze Tracking Algorithms
- Python
- Computer Vision libraries

## Detection Capabilities
- verification and identity confirmation
- Multiple face detection (detecting other people)
- Book detection
- Mobile phone detection
- Gaze direction monitoring
- Suspicious behavior alerts

## Applications
- Online examination systems
- Remote learning platforms
- Certification programs
- Corporate training assessments
